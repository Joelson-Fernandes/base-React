## 🚀 Configuração do projeto

Para instalar, siga estas etapas:

Clone o projeto e execute o comando no diretório raiz:
```
npm install
npm start
```

Sua API deverá está rodando no endereço: <a href="http://localhost:3000/">http://localhost:3000/</a>
